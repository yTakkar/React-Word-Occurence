# Usage
1. First install all dependencies:
    ```bash
    # with npm
    npm install

    # or with Yarn
    yarn
    ```

2. Then start the app.
    ```bash
    npm start
    ```

3. Open in the browser
    ```javascript
    localhost:4880
    ```
